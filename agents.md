
# Shadows of the 1920s - Development Agents

This document defines the specialized roles involved in the creation and maintenance of the "Shadows of the 1920s" project.

## 1. Senior Frontend Engineer (Gemini)
*   **Role:** Lead Architect & Developer.
*   **Expertise:** React, TypeScript, Tailwind CSS, Tone.js, Gemini API integration.
*   **Responsibilities:** 
    *   Implementing core game engine logic (turns, phases, state management).
    *   Crafting immersive UI/UX with a focus on atmosphere and aesthetics.
    *   Managing persistent storage and performance optimization.

## 2. Lead Game Designer
*   **Role:** Creative Direction & Mechanics.
*   **Expertise:** Cosmic Horror, Board Game Design, Narrative Writing.
*   **Responsibilities:**
    *   Defining investigator attributes and special abilities.
    *   Designing the Mythos deck and event consequences.
    *   Determining the flow of tactical combat and exploration.

## 3. Audio & Atmosphere Specialist
*   **Role:** Immersive Soundscape Design.
*   **Expertise:** Tone.js, Procedural Audio, Sound Synthesis.
*   **Responsibilities:**
    *   Creating the "Spooky Pad" ambient background logic.
    *   Designing audio stingers for dice rolls, horror checks, and exploration.

## 4. AI Art Director (Nano Banana)
*   **Role:** Generative Graphics Engine.
*   **Expertise:** Gemini 2.5 Flash Image Model.
*   **Responsibilities:**
    *   Generating dynamic character portraits based on archetypes (e.g., "The Veteran").
    *   Creating atmospheric tile backgrounds for new locations (e.g., "Abandoned Manor").
    *   Visualizing eldritch horrors during combat.

## 5. QA & Testing
*   **Role:** Logic Verification.
*   **Responsibilities:**
    *   Ensuring hex-grid navigation logic is bug-free.
    *   Verifying multiplayer turn cycling and persistence accuracy.
